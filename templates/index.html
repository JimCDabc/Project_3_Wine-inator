Wine-in-ator Summary

In this project we set out to accomplish two goals. The first goal is to build a model which provides an accurate idea of what chemical factors influence a higher quality of wine(Cortez et al., 2009). Our intended user for this application is a wine vintner. The second goal is to provide a user friendly page for curious Missouri wine tourists. The page provides locations of many local wineries as well as information regarding wine flavor profiles according to "Wine Folly."
